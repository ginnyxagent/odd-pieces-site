10 Core Tasks

Task 1 • Design • Owner: Martin

Price List / Sell Sheet Redesign

What: Single-page PDF retailers see first. Products, pricing, order info.

Timeline: Start Mar 25 → Done Mar 31

Front side: Logo, tagline, 2–3 hero product photos, 3 selling bullets, social proof

Back side: SKU table ($19 wholesale, $38 MSRP, case pack), order info, contact

Success: Retailer understands price, margin, how to order in <60 seconds

Task 2 • Tech • Owner: Kayl

Google Form + Auto-Reply

What: Wholesale application form. Retailers fill it out, get instant confirmation.

Timeline: Start Mar 25 → Done Mar 27

⚡ Kayl gets step-by-step guidance: Copy-paste Apps Script code, numbered clicks, everything.

What form collects: Store name, contact, email, store type, address, how they heard about us, which puzzles interest them, estimated first order qty

Success: Submit test form → auto-reply arrives in 1 min → response in spreadsheet with all fields

Task 3 • Design • Owner: Martin

Wholesale Landing Page

What: oddpieces.com/wholesale — where retailers learn about wholesale and apply

Timeline: Start Mar 28 → Done Apr 2

Page sections: Hero + "Why Carry Us" + "The Numbers" + "What You Get" + "How It Works" + CTA + FAQ

Success: Retailer lands on page, sees 50% margin opportunity, clicks "Apply" within 30 seconds

Task 4 • Ops • Owner: Hermonie

Email Templates + Gorgias Setup

What: 8 pre-written email templates for every wholesale touchpoint

Timeline: Draft by Mar 27 → Ginny approves Mar 28 → Live in Gorgias by Mar 31

Templates:

  1. Application Received (auto-reply)
  2. Application Approved
  3. Application Declined
  4. First Order Confirmation
  5. Reorder Nudge (30 days)
  6. Payment Reminder (Net 30)
  7. New Product Announcement
  8. General Inquiry Response

+ Hermonie Scripts: 5 common scenarios (discounts, returns, consignment, online sales, MAP policy)

Task 5 • Tech • Owner: Kayl

Wholesale Tracking Spreadsheet

What: Your wholesale CRM. Tracks every account from application to reorder.

Timeline: Start Mar 25 → Done Mar 27

4 tabs: Applications (auto-populated from Form) | Active Accounts | Orders | Inventory Allocation

Success: Every wholesale interaction tracked in one place. Nothing falls through cracks.

Task 6 • Design • Owner: Martin

Shelf Talker Design + Print

What: Small printed cards on shelves explaining what makes Odd Pieces special.

Timeline: Design start Mar 28 → Done Apr 2 → Print order Apr 3 → Receive Apr 10

Specs: 3.5"x2" or 4"x6" (stands), premium cardstock, brand colors, includes QR code to oddpieces.com

Print qty: 200–500 to start. Include 5–10 per retailer shipment.

Task 7 • Design • Owner: Martin

Wholesale Catalog (4–8 pages)

What: PDF catalog showcasing all wholesale SKUs with photos, descriptions, pricing.

Timeline: Start Mar 31 → Done Apr 7

Pages: Cover | Brand story + stats | Products (1 per page) | Bundles | How to order + contact

Success: Retailer flips through and says "I want to carry these."

Task 8 • Ops • Owner: Ginny

Order SOP (Step-by-Step Process)

What: How anyone processes a wholesale order without asking Ginny. This is how you scale.

Timeline: Write Mar 31 → Test with Hermonie Apr 1–2 → Finalize Apr 3

Covers: When new application arrives | When order comes in | When shipping | Monthly tasks

Success: Hermonie can process an order end-to-end without a single question

Task 9 • Docs • Owner: Ginny

Wholesale Terms & Conditions

What: 1–2 page document setting expectations. Not a legal contract — a "here's how we work together" doc.

Timeline: Draft Mar 26 → Ginny review Mar 27 → Finalize Mar 28

Covers: Pricing | MOQ | Shipping | Returns | MAP policy | Online sales | Marketing support | Account terms

Task 10 • Ops • Owner: Ginny

Inventory Allocation + Returns Policy

What: Deciding how much inventory goes wholesale vs DTC, and returns rules

Timeline: Decisions by Mar 27 → Documented by Mar 28

Allocation: Start with 20% of total inventory to wholesale. Never allocate more than you can afford to tie up in Net 30 receivables.

Returns: Damaged = full replacement. Unsold = within 90 days, original condition, 15% restocking fee.
