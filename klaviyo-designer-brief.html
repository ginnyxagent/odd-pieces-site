Template Brief

Odd Pieces · Version 1.0 · March 2026

Your job in one sentence: Build 4 email templates inside our Klaviyo account. Each one is for a different campaign type. Style them to match the brand. Save them so we can swap content in without rebuilding every time.

Section 01
Do This. Not That.
✓ Do
  • Use Playfair Display for all headlines
  • Use Inter for all body copy
  • Use yellow #F5C842 only as a thin accent (3px lines, small labels)
  • Keep CTAs black background, white text
  • Use the warm off-white #f2f0ed as the email background
  • Keep max width at 560px
  • Use inline styles only (no external CSS — Klaviyo strips it)
  • Make it fully responsive — test on mobile
✗ Don't
  • Don't use Nunito or any rounded fonts
  • Don't use full yellow backgrounds on sections
  • Don't use orange or red CTAs
  • Don't center everything — left-align body copy
  • Don't use big chunky colored buttons for every link
  • Don't add decorative emoji to headlines or body copy
  • Don't use external CSS <style> blocks for Klaviyo-specific content
  • Don't submit without testing in Gmail + Apple Mail

Section 04
The 4 Templates — Build These
A
Single Product Feature
When to use: new launch, bestseller, weekly send
Layout
Sections (top to bottom)
  • Wordmark header (centered, "Odd Pieces" in small caps)
  • Hero image — 560 × 380px, full width, 2px radius
  • 3px yellow accent line
  • Small caps label + Playfair headline + 2 body paragraphs
  • Callout box: left yellow border, light gray bg — "What's hiding in this puzzle"
  • Product card: horizontal layout — image left, name/price/CTA right
  • Star rating + italic pull quote (Playfair) + attribution
  • "You might also like" — 2 product thumbnails side by side
  • P.S. line
  • Footer (black bg)
Key specs
  • Product image in card: 200 × 150px
  • CTA button: black bg, white text, Inter 600, 12px, 0.1em letter-spacing, uppercase, 12px × 28px padding
  • Star rating: ★★★★★ Unicode — not emoji
🆕 New puzzle ⭐ Bestseller 📅 Weekly send
B
Three Product Grid
When to use: collection showcase, gift guide, re-engagement
Layout
Sections (top to bottom)
  • Wordmark header
  • Hero image — 560 × 300px
  • 3px yellow accent line
  • Small caps label + Playfair headline + 1–2 sentence intro
  • 3-column product grid — each cell: image + name (Playfair) + italic vibe tag + price + underline text link
  • Callout box: "Every puzzle has this" — left yellow border, 3 bullets
  • Dark block (black bg): "Not just a puzzle." — headline left, 4 bullet points right
  • Star rating + pull quote + attribution
  • P.S. line
  • Footer
Key specs
  • Product images: 152 × 152px, square, 2px radius
  • Product links: text underline style, not buttons — border-bottom: 1px solid #1a1a1a
  • Grid: 3 cols desktop → stacked on mobile (<600px)
🗂 Collection 🎁 Gift guide 🔁 Re-engagement
C
Bundle / Promo
When to use: discount code, bundle deal (3pk/6pk/9pk), flash sale
Layout
Sections (top to bottom)
  • Wordmark header
  • Hero image — 560 × 300px
  • 3px yellow accent line
  • Black promo bar: "Use code BUNDLE20 · 20% off at checkout" — yellow text on black
  • Small caps label + Playfair headline + 1–2 body sentences
  • Featured bundle block: image + name + what's included list + strike-through price → sale price + "You save $X" + black CTA button
  • Urgency bar: left yellow border — "Offer ends [DATE]"
  • Buy more save more tiers: 3-Pack / 6-Pack (dark, highlighted) / 9-Pack — each row has name, price, shop link
  • Star rating + pull quote
  • P.S. line
  • Footer
Tier table styling
  • 3-Pack: white bg, light border
  • 6-Pack: black bg, white text, yellow "Most Popular" label — this draws the eye
  • 9-Pack: white bg, light border, gray "Best Value" label
🏷 Discount 📦 3pk/6pk/9pk ⏰ Flash sale
D
Story / Personal Letter
When to use: welcome series, brand story, behind the scenes, re-engagement
Layout
Sections (top to bottom)
  • Wordmark header (minimal — no hero image on this one)
  • 3px yellow accent line
  • Letter body: greeting + 4–5 paragraphs, signed "— Ginny (& Terry 👋)"
  • Text CTA: not a button — plain text link with underline, e.g. "→ Puzzle Name — oddpieces.com/..."
  • Mini social proof: stars + italic quote (smaller than other templates)
  • P.S. line
  • Footer
Critical notes
  • This should feel like a personal email — no hero image, no product card, no big buttons
  • More generous padding: 48px sides (vs 40px on other templates)
  • Body copy line-height: 1.8 — more open than the others
  • Social proof section: light gray background block, not bordered
👋 Welcome 📖 Brand story 💔 Win-back
